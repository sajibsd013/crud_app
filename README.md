## Setup 

##### Step 1 - The first thing to do is to clone the repository:

```sh
git clone https://github.com/sajibsd013/crud_app.git
```



##### Step 2 - Then install the dependencies:

```sh
cd crud_app
npm install
```


##### Step 3 - Run App:

```sh
npm start
```


The Application Runs on <strong> localhost:3000 </strong>
